# 🛡️ ScamShield: AI-Powered Scam Detector

**ScamShield** is  a simple but powerful tool designed help people in Malaysia spot digital scams before it’s too late. Whether it's a suspicious SMS about your TNG e-Wallet or a weird email claiming to be from LHDN, this application utilizes AI to identify potential danger.


## 📖 Project Explanation
Digital scams are a growing crisis in Malaysia. Many users, particularly those with less technical experience, are frequently targeted by deceptive bank alerts or urgent tax-related messages. ScamShield serves as a "Safety-First" filter that does more than just identify a scam but also it breaks down the specific red flags within a message to educate users on self-protection.

## ✨ Key Features
* **Multi-Channel Scanning:** Specialized detection for Email, SMS, and URL types.
* **Tri-Color Risk Analysis:** Immediate visual feedback with High, Medium, and Low risk badges.
* **AI Confidence Score:** Displays a probability percentage generated by our machine learning model.
* **Malaysian Trust Center:** An educational hub featuring local scam trends and a direct link to the **NSRC 997** hotline.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Model:** Logistic Regression & TF-IDF Vectorization
- **Language:** Python 3.13.11

## ⚙️ Setup & How to Run
1. **Clone the Repo:**
   `git clone https://github.com/meaifa/ScamShield-AI`
2. **Install Requirements:**
   `pip install -r requirements.txt`
3. **Run App:**
   `streamlit run ScamShield.py`

## 🧠 AI Logic
The system cleans input text and uses **TF-IDF Vectorization** to highlight suspicious keywords (e.g., "Verify", "Restricted", "RM0.00"). A **Logistic Regression** model then calculates the likelihood of the content being a scam.
